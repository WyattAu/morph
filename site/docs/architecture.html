Architecture

System design and structural documentation

Architecture documents describe the high-level structure, layering, build system, and design decisions that shape the Morph project.

Specification Architecture Layers

The specification follows a layered architecture, from core definitions to validation infrastructure.

Core Types, expressions, operators, values Core.lean, Syntax.lean
Semantics Small-step reduction, evaluation, substitution Semantics.lean
Type System Typing rules, lemmas, examples Specs/TypeSystem/
Verification Progress, Preservation, ExprDepth Proofs/TypeSoundness/
Infrastructure Type checker, parser, tests TypeChecker.lean, Tests/
Validation Spec-tools (Python), 60+ spec modules scripts/spec_tools/, spec/

  scripts/                # spec-tools (Python validation)
  impl/                   # Implementation notes
